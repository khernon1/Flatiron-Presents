**Purpose:**

A Chrome extension that will allow you to test how different fonts look on your page by clicking where you want to change and choosing from a dropdown.

**Technologies used:**

Javascript and jQuery

The fonts offered are the 100 most popular fonts taken from the Google Fonts API.
